######################################################################
INSTALLATION - Points to note
######################################################################

Dell Power Manager requires two components to be installed for proper
operation: the Dell Power Manager Service and the Dell Power Manager 
Application.

Install the Dell Power Manager Service from dell.com. The installer
also installs the Dell Power Manager Application without accessing the
Microsoft Store.

######################################################################
TROUBLESHOOTING
######################################################################

Custom rules defined by your anti-virus software or installed by your
IT administrator may block successful installation/removal/upgrade of
the service. Temporarily disable anti-virus software prior to starting
the installation if you are having problems installing.
